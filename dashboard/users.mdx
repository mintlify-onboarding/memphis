### Create a new user

To create a new user, click "Add new user".

![](/dashboard/images/Screen%20Shot%202022-04-27%20at%2019.26.05.png)

![User details](/dashboard/images/Screen%20Shot%202022-04-27%20at%2019.32.49.png)

- Choose the type of user (the default is "Management").

- Add a custom password or auto-generate one.
